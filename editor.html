1

NETLIFY DROP — FASTEST

1. Go to netlify.com/drop
2. Drag your entire WebPubPortfolio folder onto the page
3. Live URL in ~30 seconds like random-name.netlify.app
4. Create a free account to lock in the URL and add a custom domain
5. To update: just drag the folder again

2

GITHUB PAGES — FREE PERMANENT URL

1. Create account at github.com
2. New repo named yourusername.github.io
3. Upload all files
4. Settings → Pages → Source: main branch → Save
5. Live at https://yourusername.github.io in ~60 seconds

3

KEEPING DATA ON A NEW DEVICE

Your edits live in this browser's localStorage.
To move to a new machine: click Export JSON Backup → save the file in your project folder → on the new device open editor.html and use the Import button below.
